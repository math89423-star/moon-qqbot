# 粟藜 · 二级模型自适应路由

纯库插件。根据 Gate 结果 + 上下文信号自动选择 LLM 模型 tier。

## 二级路由

| Tier | 用途 | 典型模型 |
|------|------|---------|
| lite | 日常闲聊、简单回复 (~85%) | deepseek-v4-pro |
| pro  | 技术问答、推理分析、交叉验证纠错 (~15%) | deepseek-v4-pro |

> 2026-06-30: JUDGE 层级 (争议仲裁) 已移除。跨 bot 行为仲裁功能删除，模型路由简化为二级。

## 升级信号

lite → pro:
- 技术领域激活 (domain awareness)
- 用户显式要求推理/分析
- 交叉验证 bot_wrong / deadlock
- Pre-flight 上下文复杂度 ≥ 4.0
- 有未解析图片
- 工具链深度 ≥ 2
- 多话题并发 (≥ 2 domains)
- 含问句 + 复杂度 ≥ 3.0

## 配置

通过管理面板 `localhost:6190` 的 LLM 槽位页配置每个 tier 的模型。

## 依赖

无（纯路由逻辑）
