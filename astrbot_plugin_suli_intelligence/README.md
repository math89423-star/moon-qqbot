# 粟藜 · 领域检测与知识注入

纯库插件。提供对话增强的 AI 基础设施。

## 模块

- **DomainDetector** — 检测用户消息涉及的知识领域（AI绘画 / 编程 / 二次元等）
- **WorldBook** — SillyTavern 兼容的世界书/Lorebook 引擎，关键词触发知识注入
- **GroupSummarizer** — 群聊话题摘要生成，周期性总结群内讨论
- **ProfileAgent** — 用户画像建档，长期行为特征蒸馏
- **PromptInterceptor** — 提示词拦截与动态注入
- **FactErrors** — 事实纠错检测

## 依赖

suli_memory, suli_emotion
