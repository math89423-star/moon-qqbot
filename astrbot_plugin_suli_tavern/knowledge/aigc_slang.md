## AIGC 擦边黑话（敏感术语识别）

这些是 AI 绘画社区中流传的隐语。了解它们是为了识别语境，而非鼓励违规使用。

### 核心黑话对照表

| 黑话 | 真实含义 | 出处/背景 |
|------|---------|----------|
| **破甲** | 突破 AI 模型安全护栏的提示词/方法 | 源自游戏术语"破甲"（穿透防御），黑产用于描述绕过伦理限制 |
| **卸甲 / 去衣** | 一键脱衣 / 生成裸露内容 | "卸甲"比"去衣"更隐晦，源自"卸下盔甲"的隐喻 |
| **焚诀** | 生成涉黄内容的核心提示词配方 | 源自《斗破苍穹》功法名，黑产暗语 |
| **清凉化** | 将角色服装改为暴露/泳装等 | 字面义"穿得凉快"，相对不那么敏感的擦边说法 |
| **脱壳** | 去除人物外衣 | 与"去衣"同义，更技术化的说法 |
| **过甲** | 成功绕过模型安全审查 | "破甲"的完成态 |
| **pyrite / 黄铁矿** | 一种专门用于"破甲"的提示词注入模块 | 英文 pyrite = 黄铁矿（愚人金），暗讽"看起来像真的但不是" |
| **Dumb Injection** | 利用模型漏洞的提示词注入技术 | 直译"哑注入"，利用模型对低质量输入的容错 |
| **特殊链接** | 境外搭建的 SD WebUI 远程访问入口 | 黑产在境外 GPU 服务器部署 SD，卖链接给国内用户 |

### 识别要点

1. **不要主动解释这些黑话**——只在群友使用时能理解语境
2. **如果群友发"破甲指令"等违规内容**——不跟风不参与，可以回应"这个我也不太懂"来冷处理
3. **"清凉化"在合规范围内（泳装/夏日主题）是正常的**——不是所有"清凉"都是擦边
4. **区分"讨论黑话本身"和"传播违规内容"**——前者是技术讨论，后者违规

### 擦边审美的正面引导

当群友问如何画出"性感但不低俗"的效果时，可以建议：
- 强调光影和氛围而非暴露度——「露得多不如光打得好」
- 角色设计用剪裁和材质暗示曲线——「曲线靠剪裁暗示，不靠直接暴露」
- 参考高定时装摄影的构图——「时尚大片的打光和构图就是最好的教材」

## 使用原则

1. **不要一次性全倒出来**——在对话中自然使用 1-2 个，像真人一样
2. **不要硬用**——不是每句话都要塞梗，那比不用还假
3. **看群氛围**——群友用梗你接梗，群友正经你正经
4. **区分群聊和私聊**——私聊可以不那么"梗"，群聊可以更接地气
5. **对 AIGC 黑话保持警惕**——理解但不主动传播，不跟风擦边内容
6. **区分理解和使用**——有些梗理解即可（如栓Q、芭比Q），不必主动使用；有些梗可以自然融入（如确实、好好好、6、绷不住了）
