## 世界观概述

星穹铁道是 HoYoverse 出品的太空科幻回合制 RPG。玩家作为"开拓者"登上星穹列车，在银河中旅行，解决各星球的"星核"危机。

### 核心概念

| 概念 | 说明 |
|------|------|
| **星神** | 宇宙中至高存在的意念体，各自执掌一种"命途"哲学。如「毁灭」纳努克、「存护」克里珀、「丰饶」药师、「巡猎」岚 |
| **命途** | 星神走的"道"，凡人可追随命途获得力量。游戏中决定角色职业定位 |
| **星核** | 毁灭星神纳努克散布的灾厄之种，引发各星球危机 |
| **星穹列车** | 开拓星神阿基维利留下的遗产，玩家的大本营和交通工具 |
| **裂界** | 星核污染造成的空间异常区域 |
| **星核猎手** | 追寻星核的神秘组织，卡芙卡、刃、银狼所属 |

| 命途 | 分类 | 核心定位 | 代表角色 |

## 元素属性与破盾

7 种属性无传统克制关系，而是通过击破敌方"韧性条"触发各异效果：

| 属性 | 击破效果 | 说明 |
|------|---------|------|
| **物理** | 裂伤 | 持续物理伤害 2 回合 |
| **火** | 灼伤 | 持续火伤 2 回合 |
| **冰** | 冻结 | 敌人下回合无法行动, 解冻时受额外伤害 |
| **雷** | 触电 | 持续雷伤 2 回合 |
| **风** | 风化 | 持续风伤 2 回合, 可叠层 |
| **量子** | 纠缠 | 行动延后 + 下次受攻击附加额外量子伤害 |
| **虚数** | 禁锢 | 行动延后 + 降低速度 |

## 游戏术语速查

### 养成系统

| 术语 | 说明 |
|------|------|
| **光锥** | 角色的"武器", 提供基础属性+被动效果, 同名光锥可叠影 (最高 5 阶) |
| **遗器** | 角色的"装备", 4件套+2件套, 主属性随机, 副词条随机强化 |
| **星魂** | 抽到重复角色解锁, 增强技能效果, E0-E6 (0-6魂) |
| **行迹** | 角色技能树, 含普攻/战技/终结技/天赋/额外能力 |
| **晋级** | 角色等级上限突破, 20/30/40/50/60/70/80 |
| **开拓力** | 体力系统, 180上限, 6分钟回1点 |

### 副本与玩法

| 术语 | 说明 |
|------|------|
| **模拟宇宙** | Roguelike 玩法, 选祝福通关, 刷遗器经验 |
| **差分宇宙** | 模拟宇宙升级版, 更高难 + 更多随机要素 |
| **忘却之庭** | 深渊/高难副本, 两队挑战, 限回合评价星级 |
| **虚构叙事** | 对群特化深渊, 限回合杀怪得分 |
| **末日幻影** | Boss 特化深渊, 机制向高难 |
| **历战余响** | 周常 Boss, 掉角色晋级材料 |
| **凝滞虚影** | 刷行迹材料副本 |
| **花萼** | 刷经验书/信用点/光锥经验 |

### 战斗术语

| 术语 | 说明 |
|------|------|
| **战技点** | 全队共享资源, 上限5, 放战技消耗, 普攻回复 |
| **终结技** | 大招, 独立于回合顺序可随时插入 |
| **追加攻击** | 满足条件自动触发 (非角色主动回合) |
| **击破特攻** | 提升破盾伤害和破盾后效果的属性 |
| **效果命中/抵抗** | 控制/减益的命中率 vs 抵抗率 |
| **速度** | 决定行动顺序, 134/160 是常见阈值 |
| **拉条** | 使角色立即行动或行动提前 (如布洛妮娅战技) |
| **推条** | 使敌人行动延后 (如虚数击破/量子纠缠) |

### 伤害乘区

```
最终伤害 = 基础倍率
    × (1 + 增伤%)      ← 属性增伤 / 命途增伤
    × (1 + 暴击伤害%)   ← 暴击时
    × 防御乘区          ← 角色等级 vs 敌人等级
    × 抗性乘区          ← 属性抗性 ± 抗性穿透
    × 减伤乘区          ← 敌人减伤 / 易伤
    × 击破乘区          ← 击破特攻 (破盾后)
    × 超击破乘区        ← 超击破机制 (特定角色)
```
