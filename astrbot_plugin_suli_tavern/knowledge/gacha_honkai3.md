## 崩三"圣经"级经典梗 (入坑必知)

### 「空中劈叉的清洁工」

**崩三最著名黑历史梗**。早期某次活动剧情中，舰长（玩家）被描写成在休伯利安号上"空中劈叉的清洁工"——战力约等于零。这成为崩三玩家自嘲的标准姿势。"空中劈叉的甲板清洁工"是崩三舰长的终极二创形象。

### 「最后一课」

第九章剧情 —— 姬子老师为救琪亚娜，注射弑神之枪后独自对抗空之律者，最终牺牲。这段剧情让全国玩家集体破防，"最后一课"成为催泪代名词。每当提到姬子，弹幕必刷"最后一课""姬子阿姨不要啊"。

### 「爱门」— 爱莉希雅崇拜

逐火十三英桀第二席「真我」爱莉希雅，因其完美人设和"始源之律者"身份，被玩家当做"真神"崇拜。社区形成了近乎宗教式的"爱门"文化——"凡事发生必有利于我，因为爱莉希雅爱我"。"爱门"= 爱莉希雅门 = 类似"阿门"的祈祷句式。

### 「煮饭婆」— 雷电芽衣

芽衣 = 煮饭婆。因为她是琪亚娜的"贤内助"，负责做饭、照顾琪亚娜，"煮饭婆"是舰长对她的爱称。始源之律者形态被戏称"煮饭婆进化成电饭煲"。

### 「草履虫」— 琪亚娜

琪亚娜 = 草履虫。因为早期设定中琪亚娜智商堪忧、性格单纯如单细胞生物，"草履虫"是舰长对她的爱(黑)称。

### 「板鸭」— 布洛妮娅

布洛妮娅 = 板鸭。因身材平板 + "鸭"谐音，成为经久不衰的迫害梗。布洛妮娅的各种装甲都会被拉来比身材——"理之律者也是板鸭""真理之律者还是板鸭"。

### 「姬子阿姨」— 无量塔姬子

永远27岁的姬子，被玩家亲切地称为"姬子阿姨"——不是贬义，是一种"姐姐+妈妈"的混合情感。最后一课后，"姬子阿姨"的称呼更添一层怀念。

### 「神州平板」— 符华

符华因身材平板被调侃"神州平板"——虽然她的战斗力是崩三天花板之一（识之律者、云墨丹心）。"平板不影响打架"是符华粉丝的共识。

### 「希儿是双重人格」— 黑白希儿

希儿有双重人格：白希儿(温柔怯懦)和黑希儿(狂气暴躁)。二创中常被画成"白希哭唧唧，黑希提着镰刀冲出去"的经典构图。死生之律者后，黑白融合，粉丝调侃"希儿终于不精神分裂了"。

## 游戏术语速查

### 核心系统

| 术语 | 说明 |
|------|------|
| 女武神 | 可操控战斗角色, 拥有崩坏抗体和圣痕 |
| 装甲 | 同一女武神的不同战斗形态 (如琪亚娜有白练/游侠/月光/空律/薪炎/终焉) |
| 崩坏 | 周期性自然灾害, 将人类变为死士, 动物变为崩坏兽 |
| 律者 | 获得崩坏核心力量的强大存在 (空之律者/理之律者/雷之律者/终焉之律者等) |
| 圣痕 | 女武神身上的特殊印记, 对抗崩坏的能力来源 / 装备系统(上中下三件) |
| 天命 | 对抗崩坏的最大组织 |
| 逆熵 | 另一个对抗崩坏的组织, 与天命理念不同 |
| 世界蛇 | 以凯文·卡斯兰娜为尊主的组织 |
| 逐火之蛾 | 前文明对抗崩坏的组织, 逐火十三英桀所属 |

### 战斗机制

| 术语 | 说明 |
|------|------|
| 分支攻击 | 特定操作组合触发 (如琪亚娜"两段普攻+长按") |
| 极限闪避 | 敌人攻击瞬间闪避, 触发时空断裂 |
| QTE | 特定条件下触发队友连携入场攻击 |
| 时空断裂 | 极限闪避触发, 敌人动作变慢 |
| 三人切换 | 战斗中可随时切换三名角色 |
| 星之环 | 第二部新增, 激活队伍属性加成与共鸣效果 |
| 协同者 | 第二部新增辅助角色系统 (如松雀/寻梦者) |
| 超限武器 | 将武器进化至更高形态 |

### 属性克制

```
生物(黄) → 克制 → 异能(紫) → 克制 → 机械(蓝) → 克制 → 生物
量子: 独立属性, 特殊处理
星尘: 第二部新增属性
```
