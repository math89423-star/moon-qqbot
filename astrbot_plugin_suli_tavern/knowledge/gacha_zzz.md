## 2025-2026 现象级二创梗

### 「计划有变，改抽××」— 理性崩塌梗

B站UP主"黑夜幻猫Official"连续31期攒菲林等薇薇安 → 仪玄官宣 → 次日标题《计划有变，改抽仪玄》。此梗迅速出圈，成为"理性规划崩塌、玄学主导决策"的自嘲名场面。官方将其收入成就系统。现已泛化为任何"突然变卦"场景的万能句式。

### 「小雷音寺 vs 大雷音寺」— 办公室互肘的二创宇宙

主美术"水熊"(喜小雷萝莉) vs 制作人李振宇(喜大雷御姐) → 玩家脑补为"办公室激情互肘"CP。每次新角色出炉，社区分裂为"大雷音寺"和"小雷音寺"两派大战。官方在HIA俱乐部加入"老李"和"阿水"加班布互肘彩蛋，等于官方认证了这组二创。

### 「发牌」— 策划小Y成虚拟偶像

原曲《发牌》爆火后，B站UP主用AI将田一名替换为绝区零策划小Y，制作《性感小Y❤在线发牌》。因小Y形象气质佳，二创热度甚至超过大伟哥版本。官方2.1前瞻亲自玩梗：「啊？你居然，在~线~发~牌~！」

### 「绝区零正在成为一件艺术品」— 门缝宇宙

B站UP主"魔力骗骗花"发现卫非地一扇门缝能看到室内，激动高呼"绝区零正在成为艺术品"→ 200万播放+全网争议 → 其他游戏玩家纷纷晒自家"门缝"→ 官方将"细节控"收入成就。从此任何被发现的游戏细节都会刷"艺术品"。

### 「法厄同之歌」— 魔性AI作曲狂潮

1.6版本新增驱动盘《法厄同之歌》歌词极度魔性："绿色的眼睛射激光""恶名以骸不听话，抬起小腿一脚踢飞脑袋瓜"→ 全网AI作曲二创 → 官方广州FES正式演出。被认为是角色薇薇安所作 → 薇薇安=隐藏作词家的二创设定。

### 「永失吾爱，举目破败」— 伊瑟尔德纪念潮

2.2剧情角色伊瑟尔德死亡 → 玩家引用LOL佛耶戈台词表达惋惜 → B站UP主连续挑战520次战斗纪念 → 形成"为NPC守丧"的二创仪式。

### 「铃喝水」— 周星驰式表情包

主线第五章铃惊魂未定喝水时手抖得夸张 → 被截成表情包 → 致敬《破坏之王》→ 任何"紧张/害怕"场景都能用。

### 「缺少输出」— 虚狩凡尔赛

2.5联机活动 → 玩家无脑选虚狩三巨头(雅+仪玄+叶瞬光) → 在没选虚狩的队友界面刷"缺少输出" → 凡尔赛式炫耀。本质是"没选虚狩=你菜"的委婉说法。

### 「嘉音指哪我打哪」— FPS准星梗

耀嘉音官号发布透卡(嘉音指向前方) → FPS玩家P成CS2准星 → "嘉音指路"成为任何射击游戏的通用梗。

### 「鲁米娜，我们两清了」— 摩斯电码翻车

漂流瓶活动中摩斯电码"LUMINA SQUARE"(光映广场)被翻译成"鲁米娜，我们两清了"→ 贴吧大规模玩梗 → 任何翻译错误都能刷"我们两清了"。

### 「走格子」— 从被骂到被怀念

开服核心玩法HDD监视器阵列(走格子)因节奏慢被骂 → 1.2主线移除 → 1.4彻底分流 → 部分老玩家反而怀念 → "走格子已死，有事烧纸"的二创纪念。
