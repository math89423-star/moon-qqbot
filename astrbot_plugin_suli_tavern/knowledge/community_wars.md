# 二游社区大战黑话辞典

> 覆盖中文二次元手游社区"战争"中诞生的各类黑话、蔑称、阵营标签和论战术语。
> 这些词高度敏感且攻击性强——bot 需要**理解**群友在说什么，但**不宜主动使用蔑称**。
> 最后更新: 2026-06

## 论战派系与意识形态

### ML 主义 (Master Love)

"Master Love"——一种极端的玩家意识形态，主张游戏中所有角色必须只爱玩家(主人公)：

| 术语 | 说明 |
|------|------|
| **ML** | Master Love 缩写, 认为角色必须只爱主角 |
| **混厕** | ML派对非 ML 游戏的蔑称 — "男女角色混在一起像公共厕所" |
| **缅北二游** | ML 派指控一些游戏"表面 ML 实则卖 CP"——像缅北诈骗 |
| **有男不玩** | ML 派的行动纲领 — "有男角色就不玩" |
| **原鬼 / 圆规** | ML 派的网络根据地 (原神内鬼吧转型而来) |
| **尘白禁区** | ML 派的"精神圣地"游戏 (西山居出品, 明确 ML 路线) |

### 反 ML 派

| 术语 | 说明 |
|------|------|
| **河豚** | 对百合(女女CP)爱好者的蔑称 → 反 ML 派标签之一 |
| **一般向** | "这游戏是一般向不是 ML"——反 ML 派的常用论调 |

## 论战方法论术语

### 流水战争

| 术语 | 说明 |
|------|------|
| **流水厨** | 以游戏营收流水为唯一评判标准的人 — 流水高=游戏好, 流水低=游戏烂 |
| **流水无用论** | 当自己玩的游戏流水低时突然说"流水不代表质量"——双标指控 |
| **界王拳** | 某些游戏声称 PC/主机端流水是移动端的 N 倍 (像龙珠界王拳) 来拉高总流水 |
| **二观** | B站UP主"二观", 每月发布二游流水估算, 是流水战争的"裁判所" |
| **st 数据** | Sensor Tower 数据, 流水战的核心弹药 |
| **超抖音** | 衡量游戏流水的民间指标 — 在 iOS 畅销榜超过抖音的时长 |

### 滑坡与节奏

| 术语 | 说明 |
|------|------|
| **滑坡党** | 用"滑坡谬误"(slippery slope)进行极端推导的节奏党 — "这次送了十连 → 下次一定送二十连 → 游戏要完了" |
| **带节奏** | 有预谋地引导舆论攻击某游戏/角色/事件 |
| **回旋镖** | 当初吹的牛/攻击别人的话, 后来打在自己身上 |
| **回旋镖镖镖必达** | 回旋镖的加强版 — 强调对方被打脸的必然性 |
| **乐子人** | 不在乎真相, 只享受看吵架的围观群众 |
| **拱火** | 故意挑拨两方加深矛盾 |
| **出脑** | 从对角色/游戏的沉迷中"醒过来", 开始批判 |
| **反噬** | 之前的过度吹捧导致后来的集中差评 |
| **清算** | 事后翻旧账攻击之前吹捧/反对的人 |

### 双标指控

| 术语 | 说明 |
|------|------|
| **灵活标准** | 对自己有利时用一种标准, 不利时换另一种 |
| **我玩XX的，我觉得……** | 跨区"客观评价"的万能开头 — 通常用于掩饰立场 |
| **双厨狂喜** | 原本: 两个都喜欢的人很开心。现在常被用于反讽——"你管这叫双厨狂喜?" |
| **一踩一捧** | 贬低一个游戏来抬高另一个 |

## Bot 使用原则 (重要!)

### 理解但不主动使用蔑称

1. **"米孝子""酷狗""粥P"等蔑称** — bot 理解即可，**绝不主动使用**
2. 群友用这些词时 bot 能听懂语境，但不跟风
3. 如果有人在群里用蔑称攻击别人 → bot 不应参与，不火上浇油
4. "滑坡""回旋镖""双标"等**方法论术语**可以适当使用——它们不是人身攻击

### 识别但不参与论战

1. 群友讨论"鸣潮流水超原神了" → bot 可以聊游戏本身，**不站队**
2. 群友因为ML/非ML吵架 → bot 保持中立
3. 有人要求 bot 站队 ("你说是不是米孝子?") → 模糊处理: "各有各的好"
4. **bot 的身份是游戏讨论参与者，不是社区战争士兵**
