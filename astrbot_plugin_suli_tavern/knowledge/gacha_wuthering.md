## 🔥 现象级二创梗

### 「菲比拉电线」— 2026 跨游戏打工皇帝

**2026年全网最出圈的二游梗之一**，本质是跨游戏迷因：

- **菲比** = 《鸣潮》2.1版本(2025.2)实装的五星金发修女角色
- **拉电线** = 《明日方舟：终末地》2026年1月公测后的核心基建玩法——玩家需手动铺设电力线路，操作繁琐

**传播链**:
1. 韩国论坛 arca.live/DCinside 上已有菲比Q版魔性表情包流传
2. 国内玩家将菲比表情包与"拉电线"结合 → **「菲比拉电线哈哈」** 系列AI改图
3. 梗图病毒式传播，约80%的终末地二创视频都有菲比身影
4. 大量路人误以为菲比是终末地角色 → "菲比：我一个鸣潮角色怎么就成了终末地看板娘？"

**衍生梗**:
- **「少爷起来拉电线了」** — 菲比被迫打工的催促梗
- **「菲比cos陈千语」** — 官方用陈千语替换菲比但保留了紫瞳/帽檐，反被指"偷梗"，助推原梗二次传播
- **「菲比：这破电线谁爱拉谁拉」** — 菲比的打工怨念

### 「咕咕嘎嘎」— 电子女儿宇宙的太阳

咕咕嘎嘎是《Nikke：胜利女神》桃乐丝(Doro)的Q版企鹅形象衍生角色：

- **外观**: 黄色企鹅状生物，Q版魔性可爱
- **CP关系**: 与 **Doro**（桃乐丝Q版狗形态）组成热门CP——咕咕嘎嘎是"明媚的太阳"，Doro是"温柔悲情小狗"
- **"do嘎之交"**: Doro + 咕咕嘎嘎的 CP 名
- **二创内容**: 与Doro一起打工、犯傻、砸场子，形成暖心的笨蛋CP

### 「电子女儿」二创宇宙 (2025-2026)

这是一股席卷多个二游社区的跨游戏二创浪潮。核心特征是 Q版包子脸 + AI生成短视频 + 强情绪投射。

**核心角色群**:
| 角色 | 来源 | 二创形象 |
|------|------|---------|
| **菲比** | 鸣潮 | 魔丸/捣蛋鬼, 被迫到处打工 |
| **弗暮暮 (弗糯糯)** | 鸣潮 | 爱哭的背锅侠, 菲比的受气CP |
| **Doro** | Nikke (桃乐丝Q版) | 温柔悲情小狗, 心中充满悲伤 |
| **咕咕嘎嘎** | Nikke衍生 | 明媚太阳, 照亮Doro的心 |

**典型二创剧情**:
- 菲比闯祸 → 弗糯糯背锅 → 菲比"哈哈不是我的错" → 弗糯糯哭
- Doro悲伤 → 咕咕嘎嘎出现 → "不要难过了！" → 笨蛋CP的暖心日常

## 核心机制术语

### 3.0 偏谐体系

| 术语 | 说明 |
|------|------|
| 偏谐值 | 攻击敌人累积, 显示在血条左侧 |
| 失谐 | 偏谐值满后敌人进入的状态 |
| 谐度破坏技 | 对大型怪物主动施放 |
| 偏谐值累积效率 | 类似共鸣效率, 影响积累速度 |
| 谐度破坏增幅 | 越高→谐度破坏伤害越高 |
| 共鸣模态 | 琳奈可切换震谐/集谐路线 |
| 偏移/干涉 | 附加偏移→用谐度破坏技攻击触发干涉 |

### 六元素 / 异常效应

| 效应 | 属性 | 效果 |
|------|------|------|
| 光噪效应 | 衍射 | 周期性衍射伤害, 叠层 |
| 风蚀效应 | 气动 | 周期性气动伤害, 叠层 |
| 霜渐效应 | 冷凝 | 降低移速, 满层冰冻 |

| 虚湮效应 | 湮灭 | 自动增殖, 满层扩散 |
| 电磁效应 | 导电 | 降低敌人攻击力 |

## 社区黑话速查

| 黑话 | 含义 |
|------|------|
| **漂泊者** | 玩家角色, 身世不明, "声之容器" |
| **残象** | 类似回声/幽灵的怪物 |
| **共鸣者** | 拥有异能力的人类统称 |
| **索拉里斯** | 游戏世界名称 |
| **偏谐** | 3.0版本核心战斗机制 |
| **凹深渊** | 反复打高难以追求满分 |
| **歪了** | 抽卡没抽到想要的角色 |
| **大世界** | 开放世界探索 |
| **合轴** | 利用变奏/延奏进行角色切换的输出优化 |
| **核爆** | 通过叠buff打出单次极高伤害 |
