# 粟藜 · 表情包发送与管理

增强插件。AI 驱动的表情包发送和图床同步。

## 功能

- **AI 自动发送** — LLM 根据对话情绪自动调用 `send_sticker` 工具
- **图床同步** — 支持 Cloudflare R2 / Stardots 等多种图床
- **类别管理** — 按情绪分类（开心/生气/害羞/惊讶 等）
- **管理页面** — 管理面板中上传/删除/分类表情包

## 配置

表情包放入 `stickers/` 目录，在 `catalog.json` 注册标签。
图床配置在管理面板中设置。

## 依赖

suli_tavern（工具调用框架）
